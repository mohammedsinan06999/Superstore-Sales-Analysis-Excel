# Superstore Sales Analysis Using Excel

## Project Overview
This project focuses on analyzing Superstore sales data using Microsoft Excel to uncover key business insights related to sales performance, regional trends, and product category contributions. The analysis is designed to support data-driven decision-making through structured reporting and pivot-based analytics.

## Objectives
- Analyze overall sales performance across different product categories
- Compare regional sales distribution
- Identify high-performing and low-performing segments
- Practice Excel-based data analysis techniques used in real-world business scenarios

## Dataset
- Source: Superstore sales dataset
- Format: Excel (.xlsx)
- Data Type: Retail transactional sales data
- Key Fields:
  - Product Category
  - Region
  - Sales

## Tools & Techniques Used
- Microsoft Excel
- Pivot Tables
- Aggregations (Sum of Sales)
- Data Cleaning
- Basic Data Transformation
- Business-oriented reporting

## Key Insights
- Furniture emerged as a major contributor to total sales
- Sales performance varies significantly across regions
- Regional analysis helps identify growth opportunities and underperforming markets
- Pivot tables provide a quick and effective way to summarize large datasets
